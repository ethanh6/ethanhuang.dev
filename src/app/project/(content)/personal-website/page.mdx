# ethanhuang.dev

Personal website and portfolio built with Next.js.

<a href="https://github.com/ethanh6/ethanhuang.dev" className="highlight-link">View source on GitHub</a>

---

## CLI usage

This site supports terminal access via `curl`.

```bash
curl https://ethanhuang.dev          # Homepage
curl https://ethanhuang.dev/resume   # Resume
curl https://ethanhuang.dev/posts    # Blog posts
curl https://ethanhuang.dev/status   # Site status
```

Download resume as PDF:

```bash
wget https://ethanhuang.dev/files/Resume-Ethan-Huang.pdf
```

---

## Tech stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Content:** MDX
- **Hosting:** AWS Amplify
- **Features:** Light/dark theme, responsive design

---

## Local development

```bash
git clone https://github.com/ethanh6/ethanhuang.dev.git
cd ethanhuang.dev
npm install
npm run dev
```
